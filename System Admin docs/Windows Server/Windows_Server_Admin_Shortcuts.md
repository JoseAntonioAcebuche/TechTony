# Windows Server Administrative Shortcuts (.msc) Reference Guide

A quick reference sheet for essential Windows Server management consoles and tools.

---

## 🔐 Active Directory & Identity
- `dsa.msc` – Active Directory Users and Computers
- `adsiedit.msc` – ADSI Edit
- `domain.msc` – Active Directory Domains and Trusts
- `gpmc.msc` – Group Policy Management
- `dssite.msc` – Active Directory Sites and Services
- `lusrmgr.msc` – Local Users and Groups
- `azman.msc` – Authorization Manager

---

## 🖥️ Core Infrastructure Services
- `dnsmgmt.msc` – DNS Manager
- `dhcpmgmt.msc` – DHCP Manager
- `certsrv.msc` – Certification Authority
- `virtmgmt.msc` – Hyper-V Manager
- `iscsicpl.exe` – iSCSI Initiator
- `fsmgmt.msc` – Shared Folders
- `dfsmgmt.msc` – DFS Management
- `dfsrdiag.exe` – DFS Replication Diagnostics

---

## ⚙️ System & Server Management
- `compmgmt.msc` – Computer Management
- `services.msc` – Services
- `diskmgmt.msc` – Disk Management
- `eventvwr.msc` – Event Viewer
- `taskschd.msc` – Task Scheduler
- `cluadmin.msc` – Failover Cluster Manager
- `devmgmt.msc` – Device Manager
- `perfmon.msc` – Performance Monitor
- `resmon.exe` – Resource Monitor
- `msinfo32` – System Information
- `cleanmgr.exe` – Disk Cleanup
- `gpupdate /force` – Refresh Group Policies
- `gpresult /r` – View Applied GPOs

---

## 🌐 Network & IIS
- `ncpa.cpl` – Network Connections
- `inetmgr` – IIS Manager
- `wf.msc` – Windows Firewall with Advanced Security
- `firewall.cpl` – Basic Firewall Settings
- `netsh` – Advanced Network Configuration
- `route print` – Routing Table
- `ipconfig /all` – Network Configuration
- `nslookup` – DNS Testing
- `ping` – Connectivity Testing
- `tracert` – Route Tracing

---

## 📊 Monitoring & Troubleshooting
- `perfmon.msc` – Performance Monitor
- `eventvwr.msc` – Event Viewer
- `resmon.exe` – Resource Monitor
- `taskmgr` – Task Manager
- `reliabilitymonitor` – Reliability History
- `mdsched.exe` – Memory Diagnostics

## 🛡️ Security
- `secpol.msc` – Local Security Policy
- `certlm.msc` – Local Computer Certificates
- `certmgr.msc` – User Certificates
- `rsop.msc` – Resultant Set of Policy
