# Logs and Troubleshooting

## Commands
```bash
journalctl -b
journalctl -p err
journalctl -u nginx
dmesg
```

## Troubleshooting Method
1. Define the symptom.
2. Check recent changes.
3. Review logs.
4. Reproduce safely.
5. Identify the root cause.
6. Apply the smallest safe fix.
7. Verify service health.
8. Document the incident.
