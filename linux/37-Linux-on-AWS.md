# Linux on AWS

## EC2 Administration
Topics include:
- Launching Linux EC2 instances
- SSH access
- Security Groups
- EBS volumes
- Elastic IP concepts
- IAM roles
- Cloud-init
- CloudWatch monitoring

## Storage
Understand EBS volume attachment, filesystem creation, mounting, and backup using snapshots.

## Security
Use least privilege, restrict inbound traffic, patch instances, and avoid exposing SSH broadly to the internet.
