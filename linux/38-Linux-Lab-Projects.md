# Linux Lab Projects

## Recommended Projects

### 1. Harden a Fedora Server
Configure users, SSH keys, firewalld, SELinux, updates, and logging.

### 2. Storage Lab
Create a partition, filesystem, persistent mount, and LVM volume.

### 3. Web Server Lab
Deploy Nginx with TLS, firewall rules, and SELinux configuration.

### 4. Backup Lab
Create an rsync backup and perform a test restore.

### 5. Monitoring Lab
Monitor Linux using Zabbix or Prometheus and Grafana.

### 6. Automation Lab
Automate user provisioning and system health checks using Bash or Python.

### 7. Ansible Lab
Manage multiple Linux systems using Ansible playbooks.

### 8. Docker Lab
Deploy a multi-container application using Docker Compose.

### 9. AWS Lab
Deploy a Linux EC2 server with secure networking and EBS storage.

### 10. Incident Response Lab
Simulate disk-full, failed service, high CPU, memory pressure, and network failures.

## Documentation Standard

For each lab document:
- Objective
- Environment
- Architecture
- Prerequisites
- Implementation
- Commands
- Validation
- Troubleshooting
- Lessons learned
