# Linux Networking

## Commands
```bash
ip addr
ip route
ip link
ss -tulpn
ping 8.8.8.8
traceroute example.com
```

## NetworkManager
```bash
nmcli device status
nmcli connection show
```

## Troubleshooting
Check link state, IP address, default route, DNS resolution, firewall rules, and service listening ports in that order.
